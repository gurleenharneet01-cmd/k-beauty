# **App Name**: GlamLens AI

## Core Features:

- Image Upload: Allow users to upload an image for analysis from the UI using Firebase Storage.
- Skin Tone Analysis: Analyze the uploaded image to determine the user's skin tone using AI to sample colors.
- Dominant Color Detection: Identify the dominant color in the uploaded image via a serverless function tool to select the color from the image.
- Color Recommendations: Provide fashion and makeup color recommendations based on the analyzed skin tone. These recommendations will come from the Cloud Function.
- Results Display: Display the analysis results, including skin tone, dominant color (with a color swatch), and color recommendations (fashion and makeup) in a clear, readable format on the UI.
- Community Page: Basic page to show community inspiration and user posts.
- Firebase Integration: Use Firebase Storage for image uploads and hosting for serving the application.

## Style Guidelines:

- Primary color: Soft pastel purple (#D0BFFF) to convey beauty and technology.
- Background color: Light gray (#F5F5F5), nearly white, for a clean and modern look.
- Accent color: Muted rose (#E0BBE4) to add a touch of warmth.
- Body and headline font: 'Belleza' sans-serif for headlines and short chunks of text and 'Alegreya' serif for longer bodies of text.
- Use minimalist icons for navigation and interactive elements.
- Employ a clean and responsive layout with a focus on image and result presentation.
- Subtle animations for loading states and transitions.
